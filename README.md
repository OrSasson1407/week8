## Project Structure

This repository contains the solution for Week 8 exercises, divided into two main parts:

 Part 2: React Application
Located in the `part2` folder.
A basic React application that includes the `Foo` component (Counter) based on the lecture slides.

**How to run (Development Mode):**
```bash
cd part2
npm install
npm start
The app will open at http://localhost:3000

 Part 3: Node Server & JWT
Located in the part3_server folder. A Node.js server using Express that serves the static React build and implements JWT authentication.

How to run:

Bash

cd part3_server
npm install
node app.js
Access points:

Static React App: http://localhost:89 (Served via Node)

JWT Login Page: http://localhost:89/login.html
